# Receipt Food using React Js

A front end web created with react.js and deploy with heroku
 
## Built with

The project was developed using :
* React js
* bootstrap
* JSX

## How to Install

* Clone Repository

### `https://github.com/dwiana161/ReceiptFood.git`

* Install the project
### `npm i`

### `npm start`
